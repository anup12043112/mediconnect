# 🏥 MediConnect: Bridging the Healthcare Gap

MediConnect is a modern, fast, and secure telemedicine platform designed to connect patients with top specialized doctors through high-quality video consultations. Built for hackathons to demonstrate a scalable healthcare solution.

## ✨ Key Features
* **Seamless Authentication:** Secure login using Google OAuth (Firebase).
* **Smart Dashboard:** Track upcoming and past appointments effortlessly.
* **Instant Consultations:** Browser-based virtual video rooms for remote doctor-patient connections.
* **Modern UI/UX:** Clean, responsive, and accessible interface built with React.

## 🛠️ Tech Stack
* **Frontend:** React.js (Vite), React Router DOM
* **Styling:** Custom Modern CSS (Google Fonts - Poppins)
* **Authentication:** Firebase (Google Auth)
* **Routing:** Client-side routing for Single Page Application (SPA) experience.

## 🚀 How to Run Locally

1. **Clone the repository:**
    Bash
    git clone [https://github.com/your-username/mediconnect.git](https://github.com/your-username/mediconnect.git)

2. **Navigate to the project directory:**
    Bash
    cd mediconnect

3. **Install dependencies:**
    Bash
    npm install

4. **Start the development server:**
    Bash
    npm run dev
